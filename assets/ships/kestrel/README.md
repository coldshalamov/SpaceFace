# SF-K0 Kestrel / BORROWED TIME

The starter ship and the first reference implementation of the SpaceFace hero-asset standard
(see `docs/Spec/GRAPHICS_STYLE_GUIDE.md` §9). It is a bespoke, authored hull — not a generic
procedural family build — that coexists with the catalog via a narrow override seam.

> **A death ship that still starts every morning.**

## Package contents

| File | Role |
|---|---|
| `kestrel_manifest.json` | Normative manifest: fiction, coordinates, dimensions, materials, sockets, metrics, contract. |
| `kestrel_reference.glb` | Engine-neutral glTF 2.0 interchange/reference model (1,844 tris, 57 meshes). |
| `kestrel_hero.svg` | Hero sheet — close three-quarter authoring reference. |
| `kestrel_blueprint.svg` | Orthographic blueprint with the seven named sockets. |

## Runtime source

The live game does **not** load the GLB. The zero-build runtime renders the Kestrel from
`src/render/ships/kestrelHero.js` (code-native Three.js geometry). The GLB is the review/DCC
interchange asset and the source of geometry metrics in the manifest. Canvas-generated decals
(the `BORROWED TIME` stencil, ghost, shark mouth, thirteen tallies) live in the runtime model
only and are documented in the manifest, not embedded in the GLB.

## Source of truth (spec §17.2)

Everything in this package is regenerated by a deterministic, zero-dependency script:

```bash
python tools/art/generate_kestrel_reference.py --output assets/ships/kestrel
```

Hand-editing a generated GLB/SVG/manifest without changing the generator creates a forked truth
and is prohibited. The generator is byte-reproducible across runs.

## Coordinate and scale contract

Right-handed; `+X` forward, `+Y` up, `+Z` starboard; metres; pivot near visual/inertial center.
Nominal dimensions 28 × 6 × 14 m; actual bounds 27.69 × 6.4851 × 13.6 m.

## Named sockets (shared by runtime + GLB)

`SOCKET_Weapon_Front`, `SOCKET_Mining_Front`, `SOCKET_Engine_Main`, `SOCKET_Utility_Dorsal`,
`SOCKET_Cargo_Ventral`, `SOCKET_Trail_Main`, `SOCKET_Camera_Focus`. See the manifest for
translations and roles. VFX and module work must consume these markers.

## Validation

```bash
npm run check:art   # GLB header/bounds/materials/sockets/size/triangles/manifest agreement
```

This is included in the normal `npm run check` chain.
